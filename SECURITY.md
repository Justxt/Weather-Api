# Seguridad

Si encuentras una vulnerabilidad, repórtala mediante un aviso de seguridad privado en GitHub:

1. Abre la pestaña **Security** del repositorio.
2. Ingresa a **Advisories**.
3. Selecciona **New draft security advisory**.

No publiques detalles explotables en un issue. El equipo revisará el reporte, confirmará el alcance y coordinará la corrección antes de divulgarlo.
